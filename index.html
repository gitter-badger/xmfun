xmfun

Yet Another Xiami Music Downloader

Installation

Add this line to your application's Gemfile:

gem 'xmfun'

And then execute:

$ bundle

Or install it yourself as:

$ gem install xmfun

Usage

xmfun
  -u url  -- url on xiami website
  -d path -- path to save your music
  -v      -- show version
  -h      -- show help message
